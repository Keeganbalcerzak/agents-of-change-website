# Hi-Fi Design Specification Targets

## Required Breakpoints
- Desktop: 1280+
- Tablet: 768-1279
- Mobile: 375-767

## Required Screens
- Home, Exam Prep, Start Trial, Success Stories, State Requirement Detail
- About, Contact, Continuing Education, Resources

## Visual Direction
- Keep evolved navy/gold identity.
- Prioritize strong type hierarchy and layered surfaces.
- Use high-motion emphasis on hero/section reveals with guardrails.

## Component Detail Expectations
- Show all key interaction states in comps.
- Include empty, loading, and error states for data/filter modules.

## Production Readiness Criteria
- Visual parity across breakpoints.
- No unresolved contrast failures.
- Reduced-motion equivalents documented.
