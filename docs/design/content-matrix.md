# Content Design Matrix

## Headline Intent Model
- Editorial: establish narrative and context
- Conversion: clarify decision and expected outcome
- Trust: reinforce credibility and risk reduction

## CTA Taxonomy
- Primary conversion: `Start Free Trial`
- Evaluation: `Compare Programs`, `See Student Outcomes`
- Support: intent-routed contact actions

## Section Messaging Rules
- Open with user context, then framework, then outcome.
- Avoid generic claims without operational explanation.
- Pair proof with mechanism whenever possible.

## Claim Consistency Rules
- Pass-rate and student-count claims must be consistent across pages.
- Compliance/approval references should appear in trust-heavy modules and footer.
